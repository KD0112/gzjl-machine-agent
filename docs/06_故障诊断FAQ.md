# 故障诊断 FAQ

本文用于回答常见挖机故障排查问题。AI 客服只能给排查方向，不能替代现场维修结论。涉及安全、拆装、压力测试、电器检测时，应建议转技术支持或维修师傅处理。

## 液压泵异响

液压泵异响可能与液压油不足、油液污染、吸油管进气、滤芯堵塞、泵内部磨损、联轴器异常、安装不到位有关。

客户问“液压泵异响怎么排查”时，应追问设备型号、异响出现时间、冷车热车是否不同、是否刚换泵、液压油和滤芯是否更换、是否有视频。不能直接判断一定是泵坏。

建议排查顺序：

1. 检查液压油液位和油液是否乳化、变黑或有杂质。
2. 检查吸油管、回油管、滤芯是否堵塞或漏气。
3. 听异响位置，是泵体、联轴器还是发动机侧。
4. 检查主泵压力和先导压力。
5. 如刚更换液压泵，确认安装、排气和油液清洁度。

## 全车动作慢或没力

全车动作慢、没力可能与液压油温高、主泵输出不足、主溢流阀压力异常、先导压力不足、滤芯堵塞、发动机功率不足有关。

客户问“PC200 全车动作慢没力怎么办”时，应补充设备型号、工作小时数、是否热车后更明显、是否单动作慢还是全车慢、是否有报警、是否近期换过滤芯和液压油。

## 发动机黑烟缺力

发动机黑烟、缺力可能与空气滤芯堵塞、喷油嘴雾化差、喷油泵异常、涡轮增压器问题、柴油质量差、进气系统堵塞有关。

客户问“发动机黑烟缺力，可能要换什么件”时，不能直接推荐喷油嘴。应先建议检查空气滤芯、柴油滤芯、进气管、故障码、喷油器回油量和发动机号。

## 行走跑偏

行走跑偏可能与左右履带张紧差异、行走马达、中心回转接头、主控阀、先导阀、油管内漏、底盘阻力差异有关。

客户问“挖机行走跑偏，左边比右边慢，先查哪里”时，应先区分空载和负载、直线行走还是转弯、左右履带松紧、是否单边无力、是否有漏油。行走马达和中心回转接头都可能相关。

## 破碎锤漏油无力

破碎锤漏油无力可能与油封磨损、氮气压力不足、管路堵塞、液压油污染、钎杆磨损、内外套间隙过大有关。

客户问“破碎锤漏油无力，应该看哪些配件”时，可提醒检查油封包、钎杆、氮气压力、油管接头、回油是否顺畅。

## 客服追问字段

故障诊断类问题建议追问：

- 设备品牌和完整型号。
- 故障出现时间，冷车还是热车。
- 是否刚维修或更换过配件。
- 故障照片或视频。
- 是否有报警码。
- 液压油、机油、滤芯是否近期更换。
- 现场维修师傅的初步判断。

## 回答边界

故障诊断回答应使用“可能原因”“建议先排查”“需要现场检测确认”等表述，不能把初步方向说成确定结论。
