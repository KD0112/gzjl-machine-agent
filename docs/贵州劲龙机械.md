## 1. 文件定位与使用方式

这份资料面向贵州劲龙机械的挖机配件销售、客服、采购和售后团队，目标不是写成百科全书，而是搭建一套可检索、可问答、可更新的销售 RAG 知识库。销售人员可以用它快速判断客户需求，客服可以用它回答常见问题，采购可以用它制定备货优先级，老板可以用它规划品类和价格策略。

- 销售接待：根据客户描述，快速追问机型、零件号、旧件照片、故障现象和工况。

- 报价辅助：按“经济型、稳定型、原厂型”三档输出，避免只报一个低价。

- 采购备货：将高频耗材、斗齿、油封、销轴、底盘件设为常备或重点备货。

- 售后风控：对液压件、电器件、密封件、定制件设置更严格的确认和退换规则。

- AI客服/RAG系统：可将每个知识块按 category、part_name、symptoms、tags 等字段切分入库。

行业背景方面，2025 年挖掘机国内和出口销量均保持增长，说明新增设备和存量维修市场同时存在机会；因此，配件销售既要做“常用件快周转”，也要做“大总成精准询价”。

## 2. 行业与客户需求分析

挖机配件客户通常不是单纯要“便宜”，而是要在价格、质量、交期和售后之间找到平衡。销售的关键是先识别客户类型，再选择报价策略。

| **客户类型**    | **主要诉求**                         | **销售应对**                                          |
|-----------------|--------------------------------------|-------------------------------------------------------|
| 个人机主/老板   | 停工损失敏感、预算敏感、希望当天发货 | 报价要快；可给经济型+稳定型两档，强调买错会耽误施工。 |
| 维修厂/修理师傅 | 懂配件但重视渠道价格和稳定供应       | 报价要专业；多问件号、总成号、旧件情况，可长期维护。  |
| 工程队/矿山客户 | 重视耐用、质保和持续供应             | 推荐耐磨型、重载型、原厂/OEM/高等级副厂；弱化极低价。 |
| 二手机/老机客户 | 价格敏感、可能接受拆车件/副厂件      | 明确寿命和售后边界；别承诺“和原厂一样”。              |
| 外贸/异地客户   | 重视包装、交期、照片、视频确认       | 需要标准化图片、尺寸、重量、包装和发票/物流信息。     |

## 3. RAG 数据结构设计

RAG 的核心不是把资料一次性塞进去，而是把知识拆成能被准确检索的小块。建议每个知识块控制在 150-350 字左右，围绕一个配件、一个故障、一个话术或一个售后规则。

| **字段**                    | **示例/说明**                     | **作用**            |
|-----------------------------|-----------------------------------|---------------------|
| knowledge_id                | 唯一编号，例如 JL-HYD-0001        | 检索、更新、溯源    |
| category                    | 配件大类：液压/底盘/发动机/电器等 | 一级过滤            |
| part_name_cn / part_name_en | 中文名/英文名                     | 中英文检索          |
| compatible_brand            | 卡特/小松/日立/三一等             | 品牌适配            |
| compatible_model            | PC200-8、CAT320D、SY215等         | 机型适配            |
| part_number                 | 零件号/OE号/供应商号              | 精准匹配            |
| symptoms                    | 客户描述的故障现象                | 故障到配件推荐      |
| recommend_logic             | 推荐逻辑：预算、工况、质保、交期  | 销售决策            |
| risk_notice                 | 安装、退换、质保风险              | 售后风控            |
| price_band                  | 参考价区间，不写死成交价          | 报价辅助            |
| source_ids                  | S1/S2等来源编号                   | 可信度与复核        |
| tags                        | 斗齿、行走慢、PC200、滤芯等       | 向量检索+关键词检索 |

推荐检索方式：关键词检索用于零件号、型号、品牌；向量检索用于客户自然语言描述，例如“挖机走偏”“热车动作慢”“大臂自己掉”。两者结合可以提高命中率。

## 5. 适配确认与询价流程

挖机配件最容易发生纠纷的原因不是价格，而是“看起来一样但装不上”。贵州劲龙机械建议执行“三图两号一工况”的询价标准。

| **确认项** | **需要客户提供**                                   | **为什么重要**                           |
|------------|----------------------------------------------------|------------------------------------------|
| 三图       | 机器铭牌照片、旧件整体照片、旧件细节/铭牌/接口照片 | 判断机型年份、安装位置、接口和外观差异。 |
| 两号       | 整机型号/序列号、零件号/OE号/总成号/发动机号       | 液压件、电器件、发动机件必须优先看号。   |
| 一工况     | 土方、石方、矿山、破碎、市政、河道等               | 决定是否推荐耐磨型、重载型或原厂/OEM。   |
| 交期       | 现货、调货、厂家发货、订做                         | 避免承诺过度；大件需确认物流和破损责任。 |
| 预算       | 经济型、稳定型、原厂型                             | 不要直接问“要便宜还是贵”，应给选择。     |

高风险配件必须二次确认：电脑板、显示屏、传感器、线束、液压泵、主控阀、行走马达、回转马达、油缸总成、回转支承、发动机大修件。

## 6. 原厂/OEM/副厂/再制造/拆车件销售策略

客户经常把“原厂、OEM、副厂、再制造、拆车件”混在一起。销售方应主动解释区别，避免因话术不清导致售后纠纷。Hitachi 官方将配件区分为 Genuine Parts 和 Remanufactured Parts，再制造件通常以较低成本提供，并强调质保；这类表达可作为销售解释的参考。

| **类型**       | **适合客户**                           | **销售注意点**                                  | **主推逻辑** |
|----------------|----------------------------------------|-------------------------------------------------|--------------|
| 原厂件/Genuine | 新机、保修期、高强度工况、客户重视稳定 | 价格高、周期可能长，但适配和品牌信任最强。      | 稳定优先     |
| OEM/配套厂件   | 客户要质量也要控制成本                 | 需确认是否真实配套体系，避免把普通副厂说成OEM。 | 性价比优先   |
| 品牌副厂件     | 老机、预算敏感、通用易损件             | 质量差距大，建议只做稳定供应商。                | 价格优先     |
| 再制造件       | 液压泵、马达、发动机、中心接头等大总成 | 必须说明旧件回收、质保边界和检测报告。          | 大修降本     |
| 拆车/二手件    | 停产机型、急用、低预算客户             | 成色寿命不确定，售后争议高，建议签确认。        | 解决稀缺     |

## 7. 主流品牌、机型和工况匹配规则

| **维度** | **内容**                                                                       | **销售判断**                                             |
|----------|--------------------------------------------------------------------------------|----------------------------------------------------------|
| 国际品牌 | Cat卡特、小松、日立、沃尔沃、神钢、斗山/迪万伦、现代、住友、加藤、久保田、洋马 | 多见于进口/合资机，零件号体系复杂，电器/液压件必须严查。 |
| 国产品牌 | 三一、徐工、柳工、临工、山河智能、龙工、山推、中联、雷沃、厦工等               | 中小挖和保有量大，常用件适合备货；国三/国四差异需注意。  |
| 按吨位   | 微挖\<6t，小挖6-13t，中挖13-30t，大挖30-50t，矿山大挖\>50t                     | 吨位直接决定斗齿、底盘、油缸、泵阀和属具规格。           |
| 按工况   | 土方、市政、矿山、石方、破碎、河道、林业、拆除                                 | 同一机型在不同工况下应推荐不同耐磨等级和质保策略。       |

推荐销售提问顺序：先问品牌和完整型号，再问年份/国三国四，再问零件号和旧件照片，最后问工况和预算。对老机和改装机，要把“需以旧件为准”写进报价备注。

## 9. 销售问答知识库与话术模板

以下内容可直接导入 AI 客服，也可作为销售微信回复模板。建议贵州劲龙机械后续把公司自己的电话、地址、质保条款和物流合作方补进去。

| **常见问题**                         | **标准回答**                                                                                                                                                |
|--------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 客户问：原厂和副厂差在哪里？         | 原厂件适配和稳定性最好，适合新机、重工况和关键总成；副厂件价格低、供货快，但质量差异大。建议根据机器年限、工况强度和预算做三档报价：原厂/OEM/高性价比副厂。 |
| 客户问：没有零件号能不能配？         | 可以初步判断，但正式报价前最好提供机器铭牌、旧件照片、安装位置、发动机号或总成铭牌。尤其是电器件、液压泵、马达、主控阀，不能只看机型。                      |
| 客户问：为什么同一个斗齿价格差很多？ | 斗齿价格主要受材质、热处理、重量、齿型、适配系统和工况影响。土方齿便宜，岩石齿/矿山齿更耐磨但价格更高。                                                     |
| 客户问：液压泵副厂能不能用？         | 老机或预算有限可以用高等级副厂/再制造；长期高强度工程建议优先原厂/OEM或有试验台检测的再制造。安装前必须清洗油路、换液压油和滤芯，否则容易产生售后争议。     |
| 客户问：你们价格能不能再低？         | 可以按三种方案给您选：经济型、稳定型、原厂型。低价方案能降低采购成本，但关键件我建议至少选质保明确的稳定型，避免停工损失更大。                              |
| 客户问：买错了能不能退？             | 常规未安装件可以按公司规则处理；电器件、液压件、密封件、定制件、已装机件和有油污/磨损痕迹件需要先检测确认，发货前请务必核对照片和零件号。                   |

## 11. 售后质保与风险控制

- 发货前：核对零件号、照片、数量、包装、客户收货地址和是否含税/运费。

- 安装前：提醒客户先对比旧件、新件、接口、孔位、插头、齿数、管口方向。

- 液压件：安装前建议清洗油路、更换液压油和滤芯；否则污染导致损坏容易产生争议。

- 电器件：插头针脚、程序版本、报码情况必须确认；已通电装机件退换风险高。

- 密封件：油封、O型圈、修理包开封和装机后一般难以二次销售，需明确规则。

- 大件：驾驶室玻璃、油缸、大臂、铲斗、履带等需确认物流破损责任和签收检查。

| **售后类别**   | **典型配件**                                                 | **处理建议**                                         |
|----------------|--------------------------------------------------------------|------------------------------------------------------|
| 可正常协商     | 未装机、未损坏、包装完整、非定制常规件                       | 按公司退换规则处理。                                 |
| 需检测后处理   | 液压泵、马达、主控阀、油缸、电磁阀、传感器                   | 判断是否安装污染、油路问题、电路问题或配件本身问题。 |
| 原则上谨慎退换 | 电脑板、显示屏、线束、定制件、切割件、油封修理包、玻璃破损件 | 报价前提前写明。                                     |
