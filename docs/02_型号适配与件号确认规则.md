# 型号适配与件号确认规则

本文用于回答客户关于“能不能配”“通不通用”“没有零件号能不能确认”的问题。适配类问题风险较高，原则是：能根据经验给方向，但不能只凭一句话承诺一定能装。

## 适配判断优先级

适配判断建议按以下优先级：

1. 旧件铭牌、零件号、总成号。
2. 旧件多角度照片，包括安装面、接口、油口、插头、固定孔位。
3. 设备品牌、完整型号、年份、发动机号或机身铭牌。
4. 故障现象和安装位置。
5. 客户口述简称或模糊型号。

如果客户只提供“PC200”“320D”“SY215”这类机型，客服可以先说明可能适配范围，但应继续追问旧件号或照片。

## 常见机型说明

小松 PC200 是中型挖机常见型号，客户常问液压泵、回油滤芯、喷油嘴、斗齿、行走马达等。PC200 不同年份、不同发动机配置、不同液压泵编号可能存在差异。

卡特 320D 常见咨询包括主控阀、液压泵、滤芯、行走马达、传感器。卡特机型中 CAT、卡特、卡特彼勒常被混用，客服应统一识别为卡特品牌。

三一 SY215 常见咨询包括液压泵、主控阀、滤芯、斗齿、油缸。国产品牌同一机型也可能因年份和配置不同产生配件差异。

日立 ZX200 常见咨询包括行走马达、回转马达、液压泵、喷油嘴、滤芯。日立部分配件需要结合旧件号确认。

## 旧件照片要求

当客户没有零件号时，可以要求提供以下照片：

- 旧件整体照片，能看出外形和安装位置。
- 铭牌、钢印、标签或二维码照片。
- 接口、插头、油口、孔位照片。
- 损坏部位照片。
- 设备铭牌或发动机铭牌照片。

如果照片模糊、缺少铭牌或只拍局部，客服不能直接确认适配，应要求补拍。

## 发动机号与喷油嘴匹配

客户问“发动机号可以配喷油嘴吗”时，可以回答：发动机号对喷油嘴匹配有帮助，但最好同时提供喷油嘴旧件号、发动机铭牌和旧件照片。不同发动机配置或排放阶段可能导致喷油嘴型号不同。

如果客户只有挖机型号，没有发动机号和喷油嘴编号，客服应提示只能先给可能方向，不能保证直接安装。

## 通用件与非通用件

滤芯、斗齿、销轴等常用件有一定通用空间，但仍要确认尺寸、接口或齿座型号。液压泵、主控阀、电脑板、电磁阀、喷油器等高风险件不建议说“通用”，应以件号和实物为准。

电器件尤其不能只看外形。插头相同不代表程序、参数或电压一定一致。

## 客服回答模板

客户问“这个能不能配 PC200”时，可以回答：

> 仅凭 PC200 这个机型还不能百分百确认。PC200 不同年份和配置可能用不同件号。您可以发一下旧件照片、铭牌号或零件号，我先帮您按型号和安装位置核对，再给您确认原厂、副厂或经济型方案。

客户问“没有零件号能不能配”时，可以回答：

> 没有零件号也可以先判断方向，但需要旧件照片、安装位置和设备铭牌。高价值液压件和电器件建议一定核对件号，避免买错后耽误安装。

## 风险边界

适配类问题不能只靠模型或客服经验直接承诺。正式下单前，建议由人工客服根据照片、件号、库存批次和供应商确认结果复核。
