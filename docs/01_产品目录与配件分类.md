# 产品目录与配件分类

本文用于补充企业客服 RAG 的产品目录知识，帮助客服在客户只说“泵”“阀”“滤芯”“马达”这类简称时，快速判断可能涉及的配件类别、常见叫法、确认字段和风险点。

## 液压系统配件

液压系统是挖掘机动作执行的核心，客户常问的配件包括液压泵、主控阀、行走马达、回转马达、中心回转接头、分配阀、液压油缸、油管、先导阀、电磁阀、液压油滤芯等。

液压泵也常被客户称为主泵、泵总成、大泵。询价前必须确认设备品牌、完整型号、旧泵铭牌、泵壳编号、是否原厂或副厂、是否接受再制造件。液压泵属于高价值件，不建议只凭“PC200 液压泵”直接报价。

主控阀也叫多路阀、分配阀。确认时要看设备型号、阀体铭牌、阀组数量、是否带电磁阀、是否有改装。主控阀适配风险较高，安装位置和油口布局不一致会导致无法安装。

行走马达和回转马达常被客户混称为马达。客服应追问是行走无力、跑偏，还是回转慢、回转异响。不同位置的马达不能简单通用。

## 发动机与燃油系统配件

发动机相关配件包括喷油嘴、喷油器、喷油泵、启动马达、发电机、涡轮增压器、水泵、皮带、机油泵、缸垫、活塞环、滤清器等。

喷油嘴和喷油器在客户口语中经常混用。匹配喷油嘴时，发动机号、喷油嘴编号、发动机品牌和排放阶段都很重要。只提供挖机型号时，仍建议客户补充发动机铭牌或旧件照片。

涡轮增压器询价时要确认发动机型号、旧件号、是否烧机油、是否异响、是否叶轮损坏。该类配件安装后出现故障时，需要区分配件质量问题和发动机进排气系统问题。

## 底盘与工作装置配件

底盘件包括履带链条、履带板、支重轮、托链轮、引导轮、驱动齿、涨紧油缸、履带螺栓等。工作装置包括斗齿、齿座、销轴、衬套、连杆、铲斗、大小臂油缸等。

斗齿常见问题是客户只说“要斗齿”，但不同齿座型号差别很大。客服应追问斗齿型号、齿座照片、机器吨位、工况是土方还是矿山。

履带链条和支重轮要确认机型、链节数、板宽、左右是否成套、是否重载工况。底盘件体积重量较大，物流和包装需要单独确认。

## 电器与传感器配件

电器件包括电脑板、显示屏、控制器、线束、压力传感器、转速传感器、油温传感器、电磁阀、继电器、保险盒等。

电器件退换风险高，通电、插装、拆封后一般不能直接按普通件退换。客服在售前应提醒客户先核对件号、接口、插头针脚、旧件照片和故障码。

## 保养耗材

保养件包括空气滤芯、柴油滤芯、机油滤芯、液压油滤芯、油水分离器、机油、液压油、齿轮油、防冻液等。

滤芯类适合做常备库存，但仍需要确认设备型号、发动机型号和滤芯编号。客户问“滤芯一套多少钱”时，要明确是一套几只，包含空滤、柴滤、机滤、液压滤还是回油滤芯。

## 客服确认字段

所有配件询价至少建议确认以下字段：

- 设备品牌和完整型号，例如小松 PC200、卡特 320D、三一 SY215。
- 配件名称和旧件照片。
- 零件号、铭牌号或总成号。
- 品质档位：原厂、副厂、经济型、再制造或拆车件。
- 数量、是否急用、收货城市。
- 是否安装过、是否需要质保、是否接受人工复核后报价。

## 回答边界

客服可以给出配件类别、常见叫法和需要补充的信息，但不能在关键信息不足时承诺适配、价格、库存或售后结果。高价值液压件、电器件、发动机件必须人工二次确认。
